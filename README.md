# StudyMatch (Student Group Matching System)

This is a short Java system for basic management of a Student Group Matching System. It can record Students and Groups, and perform the logic necessary to register Students to their Groups. I made it as a personal practise project for my University Java Class. I asked Google Gemini to make a Java system activity for me to practise, and this is the system I made for it.

The instructions sheet is uploaded to this repository. It is, however, in Brazilian Portuguese.
